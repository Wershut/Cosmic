Diseña una página web con un formulario que contenga un campo de tipo "color" y un select con tres opciones: "Elija una categoría", "Electrodomésticos" y "Ropa". El campo de color debe estar deshabilitado por defecto.

Cuando el usuario seleccione la opción "Electrodomésticos", el campo de color debe habilitarse y permitir al usuario elegir un color. Si el color seleccionado es distinto de verde (#00ff00) o azul (#0000ff), no se deben enviar los datos del formulario.

Además, la página debe contener un segundo select con algunas marcas de electrodomésticos y otro select con algunas marcas de ropa. Inicialmente, el segundo select debe tener la propiedad CSS "visibility:hidden". Solo si la opción elegida en el primer select es "Electrodomésticos", se debe mostrar el segundo select con las marcas de electrodomésticos. Si la opción elegida es "Ropa", se debe mostrar el tercer select con las marcas de ropa.

Requisitos:

  • Utilizar HTML, CSS y JavaScript para crear la página web.
  • El formulario debe contener un campo de tipo "color" y un select con tres opciones: "Elija una categoría", "Electrodomésticos" y "Ropa".

  • Cuando el usuario seleccione la opción "Electrodomésticos", el campo de color debe habilitarse y permitir al usuario elegir un color.
  • Si el color seleccionado es distinto de verde (#00ff00) o azul (#0000ff), no se deben enviar los datos del formulario.
  • La página debe contener un segundo select con algunas marcas de electrodomésticos y otro select con algunas marcas de ropa.
  • Inicialmente, el segundo select debe tener la propiedad CSS "visibility:hidden".
  • Solo si la opción elegida en el primer select es "Electrodomésticos", se debe mostrar el segundo select con las marcas de electrodomésticos. Si la opción elegida es "Ropa", se debe mostrar el tercer select con las marcas de ropa.
